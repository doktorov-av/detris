# Detris

This is a good old tetris game written in React & Vite & TypeScript

# Why another one?

I've been coding for a while (2+ years in total) in C++ & Go and had to learn Frontend development, so i wrote this project for practce

## About code

This project can demonstrate good coding practices such as:

- Single responsibilty: each class/component has it's purpose
- Reusage: if some code is duplicated, it goes into separate class/function
- Clean structure (mostly): it is easy to locate file that is responsible for specific game mechanic
- Tuning: it is easy to vary game properties from /src/tuning folder

I follow these practices all along my other projects, so hope it helps you better understand React!
